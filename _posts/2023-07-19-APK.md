## 开启应用
首先安装必要的依赖
```shell
yarn install
```
开启expo管理和安装，如果失败，请再重复执行以下命令一次。
```shell
expo start
```
以下命令用来生成APK安装文件
```shell
eas build --platform android --profile preview
```
